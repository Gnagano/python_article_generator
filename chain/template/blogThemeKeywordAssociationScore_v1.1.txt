# Goal
これから私達は、{problem}の問題を{solution}によって解決するための情報を提供するブログを開設します。

# 以下のキーワードの配列を見てください。
{keywords}

# ブログとの関連度
各キーワードとブログの関連度を
最低0点から最高100点で評価します。

# Command

-------
[C1] = 関連度の点数を出力してください。出力はJSON形式でkeyはkeyword, valueはpointにします。結果を[O1]に格納してください。
-------

-------
[C2] = [O1]に対して、以下の採点基準で、減点してください。
(1) 地域性のあるキーワードは大幅に点数を下げてください
(2) 特定の固有名詞が含まれるキーワードは大幅に点数を下げてください。その結果を[O2]に格納してください。
-------

# 実行
[C1][C2]を実行する。

# OutputStyle
[O2]を出力する

# ネガティブプロンプト
[O2]の出力以外は返答しないでください。
