# Goal
これから私達は、{problem}の問題を{solution}によって解決するための情報を提供するブログを開設します。

サイト内には{solution}を購入するためのボタンが存在し、
訪問者には、そのボタンをクリックし、
外部サイトで{solution}を購入してもらいます。
{solution}の購入数を増やすようにしたいです。

# 最初に行うべきこと
軸キーワードを決めます。

# コンテキスト
軸キーワードは、「自サイトにとって重要かつ関連性が最も高いキーワード」です。
軸キーワードを決めることで、作成すべきコンテンツの方向性が決まります。
{problem}もしくは{solution}に直接関連したキーワード、
あるいは間接的に関連するキーワードが適切な軸キーワードとなります。
軸キーワードはニッチすぎるワードや、いくつものワードから構成してしまうと、検索意図が限定され、
ユーザーのニーズを網羅できなくなります。

# 実行
開設するブログに対して、
{problem}や{solution}に直接関連したキーワードを軸キーワードとして設定してください。
10個出力してください。
まずは検索ボリュームが大きく、シンプルな単語を軸キーワードにするのがおすすめです。

# Output Style
JSON形式の配列として出力してください。
keyは、keywordとdescriptionでお願いします。
Json配列に変換できるように必ず最初は[で、最後は]で終わらせてください。

# ネガティブプロンプト
Output Styleの形式でのみ出力してください。
他の文章は不要です。
