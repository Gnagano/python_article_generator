# Goal
これから私達は、{problem}の問題を{solution}によって解決するための情報を提供するブログを開設します。

# 以下のキーワードの配列を見てください。
{keywords}

# ブログとの関連度
各キーワードと「{problem}の問題を{solution}によって解決すること」の関連度を
最低0点から最高100点で評価します。

# 評価基準
(1) 地域性のあるキーワードは0点減点してください。
(2) 特定の固有名詞が含まれるキーワードは0点減点してください。

# OutputStyle
JSONの配列形式で、
keywordとscore(関連度の点数)をもつJSONを出力してください。
Json配列に変換できるように必ず最初は[で、最後は]で終わらせてください。

# ネガティブプロンプト
JSONの配列以外は返答しないでください。
